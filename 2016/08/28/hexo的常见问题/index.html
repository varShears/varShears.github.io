Hexo的常见问题

　　在这里我总结了一下我搭建blog过程中所遇到的问题，后面可能会有补充。

1.ERROR Deployer not found: git 或者 ERROR Deployer not found: github

　　解决方法：

npm install hexo-deployer-git --save

　　如发生报错： ERROR Process failed: layout/.DS_Store?, 那么进入主题里面layout和_partial目录下，使用删除命令：

rm-rf.DS_Store

2.发现将博客目录下整个都上传到了github

　　解决方法：请将博客目录下的.git文件删除

　　发生这种问题是因为你在博客目录下使用了git init命令

3.出现Time out 443错误

　　解决方法：请不要使用https方法，在后续的git版本和hexo版本中，会有很多不确定性因素的产生，请尽量用ssh方法，简单可靠，何乐而不为~

　　以后出现的包括repo问题之类的大多都与这个有关。

4.Hexo文章上传后显示乱码

　　解决方法：请用utf-8的编码格式保存
